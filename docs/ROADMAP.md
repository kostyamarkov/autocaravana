**Этапы (Milestones)**

1) Минимальный рабочий продукт (MVP) — базовый набор (priority: Высокий)
   - Цели:
     - Структурировать `js/data.js` (примерная схема ниже).
     - Рендер списка элементов и детальной вьюхи в `index.html` через `app.js`.
     - Простейшая галерея (клик на миниатюру открывает модальное окно с навигацией).
     - Hash-маршрутизация для прямых ссылок.
   - Оценка: 1–2 дня.
   - Зависимости: нет внешних библиотек, можно сделать только на нативном JS.

2) Поиск и улучшения данных (priority: Высокий)
   - Цели:
     - Подключить Fuse.js и индексировать `title`, `description`, `tags`.
     - Добавить UI поля поиска с автокомплит/результатами "на лету".
     - Улучшить структуру данных для удобной фильтрации.
   - Оценка: 0.5–1 день.
   - Зависимости: Fuse.js (клиентская библиотека).

3) Галерея — реализована (✅ ВЫПОЛНЕНО)
   - ✅ Кастомная модальная галерея для просмотра увеличенных изображений.
   - ✅ Все 44 изображения кликабельны с визуальной обратной связью (hover effect).
   - ✅ Свайпы в модальном окне (влево/вправо) для переключения фото на touch/pen-устройствах.
   - ✅ Модальное окно закрывается через X кнопку, клик вне изображения, или клавишу ESC.
   - ✅ Размеры изображения адаптивны и привязаны к ширине текстовой части через CSS переменные.
   - Опционально (future): добавить навигацию между слайдами в модальном окне, full-screen режим, zoom с мышью.

3.5) Мобильная адаптация — реализована (✅ ВЫПОЛНЕНО, ветка 1.0m)
   - ✅ Адаптивное меню с различным поведением для десктопа и мобильных устройств.
   - ✅ Десктоп: меню развернуто по умолчанию, сворачивается до 70px с иконками, состояние сохраняется в localStorage.
   - ✅ Мобильная: меню свернуто по умолчанию (70px полоска), разворачивается до 90% с темным overlay.
   - ✅ Иконки для всех пунктов меню (SVG/PNG в `pictures/icons/`).
   - ✅ Tooltips для свернутого меню на десктопе (отключены на мобильных).
   - ✅ Скроллинг меню на маленьких экранах (десктоп и мобильные) без видимого scrollbar.
   - ✅ Touch-поддержка: плавная прокрутка с инерцией (`-webkit-overflow-scrolling: touch`).
   - ✅ Полноэкранный скроллинг: перенесен с `.section-content` на `.main-content` для единого scrollbar.
   - ✅ iPhone scroll rollback: отключён специальный нативный document-scroll режим для iPhone, используется единая контейнерная прокрутка как в текущей мобильной реализации.
   - ✅ Автоматическое закрытие мобильного меню при выборе пункта или клике на overlay.
   - ✅ Переключение между режимами при resize окна.
   - ✅ Протестировано на разрешениях от 428x926 (iPhone) до 4K (3840x2160).
   - Детали: см. раздел "Мобильная адаптация (Ветка 1.0m)" в [TECHSTACK_AND_PAGES.md](TECHSTACK_AND_PAGES.md).

4) Локализация (priority: Средний)

5) Опционально: сборка и CI (priority: Низкий)
   - Цели:
     - Настроить простую сборку/минификацию (esbuild/parcel) и проверку линтером.

   - Оценка: 1 день.

**Пример структуры записи в `js/data.js` (рекомендация)**

```
{
  id: "kitchen",
  title: { ru: "Кухня", en: "Kitchen" },
  description: { ru: "Описание...", en: "Description..." },
  tags: ["газ", "раковина"],
  images: ["img/kitchen-1.jpg", "img/kitchen-2.jpg"]
}
```

**Кто и что делает (роли / ownership)**

- Разработчик интерфейса: реализует рендер, маршруты, интеграции (Fuse.js, галерея).
- Контент-менеджер: заполняет `js/data.js`/`locales/*.json` содержимым и фотографиями.
- Тестировщик / QA: проверяет работу поиска, галереи и переключение языка.
