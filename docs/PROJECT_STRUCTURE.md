# Структура проекта

Полная структура файлов и папок проекта "Инструкция для пользователя автодома".

```
autocaravana/
├── index.html                 # Главная HTML страница (точка входа)
├── css/
│   └── style.css             # Все стили приложения (десктоп и мобильная адаптация)
├── js/
│   ├── app.js                # Основная логика приложения
│   ├── data.js               # Данные контента (описания элементов автодома)
│   └── version.js            # Управление версией приложения
├── pictures/
│   ├── camperPhotos/         # Фотографии автодома (image1.png - image44.png, все в PNG)
│   └── icons/                # Иконки для пунктов меню (SVG/PNG)
├── docs/                     # Документация проекта
│   ├── PROJECT_STRUCTURE.md  # Структура проекта (этот файл)
│   ├── ROADMAP.md            # Дорожная карта и статус выполнения задач
│   ├── TECHSTACK_AND_PAGES.md # Описание стека технологий и страниц
│   ├── VERSION_MANAGEMENT.md # Управление версиями и Git workflow
│   └── version.txt           # Текущая версия (дата + commit hash)
└── scripts/
    ├── setup-hooks.ps1       # Установка Git hooks
    └── update-version.ps1    # Автоматическое обновление версии
```

## Описание ключевых файлов

### HTML
- **index.html** — единственная HTML страница приложения. Содержит:
  - Боковое меню (sidebar) с навигацией
  - Основную область контента
  - Модальное окно для просмотра изображений
  - Поле поиска

### CSS
- **css/style.css** — все стили интерфейса:
  - CSS переменные для цветовой схемы
  - Стили для десктопной версии (>768px)
  - Адаптивные стили для мобильной версии (≤768px)
  - Стили модального окна галереи
  - Адаптивная сетка для изображений

### JavaScript
- **js/app.js** — основная логика:
  - Инициализация приложения
  - Управление состоянием (state)
  - Рендеринг меню и контента
  - Поиск по ключевым словам
  - Управление модальным окном
  - Обработка событий (клики, resize, keyboard)
  - Адаптивное поведение для мобильных/десктопа

- **js/data.js** — данные приложения:
  - Массив объектов с описаниями элементов автодома
  - Структура: { id, title, description, content (HTML) }
  - Встроенные изображения в HTML контенте

- **js/version.js** — управление версией:
  - Отображение версии в футере меню
  - Синхронизация с version.txt

### Изображения
- **pictures/camperPhotos/** — 44 PNG изображений автодома:
  - Именование: image1.png, image2.png, ... image44.png
  - Используются в контенте через `<img class="content-image">`
  - Все изображения кликабельны (открывают модальное окно)
  - Все изображения переведены в PNG формат (из исходных WebP)

- **pictures/icons/** — иконки для меню:
  - SVG или PNG иконки для каждого пункта меню
  - Размер: 20x20px (развернуто), 24x24px (свернуто)
  - Используются через `<img class="menu-icon">`

### Документация
- **docs/PROJECT_STRUCTURE.md** — этот файл, структура проекта
- **docs/ROADMAP.md** — дорожная карта разработки и статус задач
- **docs/TECHSTACK_AND_PAGES.md** — подробное описание технологий и архитектуры
- **docs/VERSION_MANAGEMENT.md** — система управления версиями
- **docs/version.txt** — текущая версия в формате "YYYY-MM-DD commit_hash"

### Скрипты
- **scripts/setup-hooks.ps1** — PowerShell скрипт для установки Git hooks
- **scripts/update-version.ps1** — автоматическое обновление версии после коммита

## Ветки Git

- **main** / **1.0** — стабильная десктопная версия
- **1.0m** — мобильная адаптация (текущая рабочая ветка)
  - Адаптивное меню с различным поведением для десктопа/мобильных
  - Скроллинг меню на маленьких экранах
  - Touch поддержка и оптимизация для мобильных устройств

## Зависимости

Проект не использует внешних зависимостей:
- Чистый HTML5
- Нативный CSS3 (без препроцессоров)
- Vanilla JavaScript (без фреймворков)

## Запуск проекта

```bash
# Запуск локального HTTP-сервера (Python)
python -m http.server 8000

# Затем открыть в браузере:
# http://localhost:8000/
```

## Размер проекта

- HTML: 1 файл (~3 KB)
- CSS: 1 файл (~20 KB)
- JavaScript: 2 файла основных (~15 KB) + 1 файл версии (~1 KB)
- Изображения: 36 PNG файлов (~2-5 MB)
- Иконки: ~20 SVG/PNG файлов (~100 KB)
- Документация: ~5 MD файлов (~50 KB)

**Общий размер:** ~3-6 MB (зависит от размера изображений)
